# Getting started

### Scaffold new project

::code-group
  ```bash [Yarn]
  yarn create svarta-app my-app
  ```
  ```bash [NPM]
  npx create-svarta-app my-app
  ```
  ```bash [PNPM]
  pnpm create-svarta-app my-app
  ```
::

### Start development server

::code-group
  ```bash [Yarn]
  yarn dev
  ```
  ```bash [NPM]
  npm run dev
  ```
  ```bash [PNPM]
  pnpm run dev
  ```
::
