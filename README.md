## Bootstrap new project

```bash
pnpm create svarta-app
yarn create svarta-app
npm create svarta-app
```

For development, run:

```bash
pnpm dev
yarn dev
npm run dev
```

For building, run:

```bash
pnpm build
yarn build
npm run build
```
